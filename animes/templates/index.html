Anime API

Easy authentication and authorization in Flask.

It creates a new sqlite (local FS DB) and populates it from data/anime.csv.

Methods:

/all will output first 100 animes, sorted by their anime ID.

/anime/[anime name] lists all [anime name]'s details. Also any anime that contains [anime name] in its name.

"/anime/add?[anime_id=str&name=str&type=str&genre=str&episodes=int&rating=str] adds a new entry if absent (and/or lists it if already present).

/anime/update?anime_id=int[¶ms] will update anime_id's params.

/anime/delete?anime_id=int deletes anime_id from the DB.

How to run it:

export FLASK_APP=animes flask run
